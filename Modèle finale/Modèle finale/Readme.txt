Ce dossier contient les meilleures modèles d'apprentissages CNN,BLSTM,MLP,CNN-BLSTM pour chaque ensemble de données utilisés dans les expérimentations. Pour chaque modèle, nous avons attribué trois fichiers :
_________________
Fichier weights : qui contient les poids du modèle.
Fichier historique : qui contient l'historique d'exécution du modèle.
Ficher  résultat : qui contient l'évaluation du modèle avec les métriques (LOSS, ACCURACY, RECALL, PRECISION, FMEASURE).
